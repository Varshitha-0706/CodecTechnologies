# Employee Attrition Analysis

## Overview
This project analyzes and predicts employee attrition to help HR teams reduce turnover and improve workforce planning.

## Dataset
- IBM HR Analytics Attrition Dataset (Kaggle)  
- https://www.kaggle.com/datasets/pavansubhasht/ibm-hr-analytics-attrition-dataset

## Tools
Python, Pandas, Scikit-learn, Power BI

## Approach
- Data preprocessing and EDA  
- Random Forest classification  
- Feature importance analysis

## Key Insights
- Income, age, and overtime strongly affect attrition
- Early-career employees leave more often

## Outcome
Provides data-driven insights for HR retention strategies.
